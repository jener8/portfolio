# Portfolio Image Reference Guide

## How to Change Images

1. **Select the image** in v0's preview using element selection
2. **Look at the code panel** on the right - you'll see `src="/some-image.jpg"`
3. **Upload your new image** using: ```png file="public/my-image.png" url="BLOB_URL"```
4. **Tell me which image to replace** (e.g., "replace i-Kfz Image 2 with /my-image.png")
